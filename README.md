# LEO - Mobile Prototypes

A high-fidelity mobile prototype for the LEO logistics platform.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  **Open in your browser**:
    Navigate to [http://localhost:3000](http://localhost:3000).

4.  **Mobile View**:
    Open Chrome DevTools -> Toggle Device Toolbar -> Select "iPhone 14 Pro" or similar.

## Features

- **Fibonacci Intro**: Custom intro animation.
- **Courier App**: Onboarding, Route Planning, Stop Details.
- **Customer App**: Live Tracking, Plan B Selection.
- **Design System**: "LEO" Navy/White theme with Inter font.

## Technologies

- Next.js 14
- Tailwind CSS 3
- Framer Motion
- Lucide Icons
